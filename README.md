# 2. Summary

**Purpose:**  
This dataset explores how AI tools influence emotions, identity, authenticity, creativity, originality, meaning, and inner cognitive experience.

**Scope:**  
Adults aged 18+, across the UK, US, Canada, Australia, New Zealand, Ireland, and other English-speaking regions.  
Data collected November 2025 via Prolific.  
Final sample size: **n = 507**.

**Themes:**  
Emotional change · Identity · Creativity · Authenticity · Values · Meaning · Self-perception · Cognitive influence · AI–human interaction

**Outputs:**  
Numeric Likert-style scores (1–7), open-text reflections, and demographic variables.

**Data Integrity Note:**  
This dataset has been fully cleaned, anonymised, and verified according to HCI’s open-data protocol.  
A SHA-256 checksum file is included to ensure long-term data integrity.

# 7. Method

**Design:**  
Cross-sectional online survey.

**Instrument:**  
33 × Likert items (1–7)  
2 × open-text questions  
Demographics: age group, gender, work status, country, AI-tool use.

**Sampling:**  
Prolific.  
Global English-speaking adults.  
Collection: November 2025.

**Ethics:**  
Informed consent obtained.  
Data anonymised.  
Participants could withdraw before submission.

# 9. Processing & Cleaning

**Pipeline:**  
1. Exported raw responses from Google Forms  
2. Verified consent  
3. Checked and resolved duplicate Prolific IDs  
4. Verified complete records  
5. Confirmed Likert numeric ranges  
6. Checked for straight-lining (none detected)  
7. Renamed headers to snake_case  
8. Validated demographic categories  
9. Prepared README + data dictionary  
10. Generated SHA-256 checksums
